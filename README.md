# Telegram Userbot (зеркало аккаунта через терминал)

Скрипт входит в твой **личный** аккаунт Telegram через официальный API (MTProto)
и позволяет читать/писать чаты прямо из консоли.

## Установка

```bash
cd /Users/kamilimangulov/yoy
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

Есть два варианта интерфейса — терминал и веб-панель. Оба используют один и тот же
файл сессии `user_session.session`, так что авторизацию достаточно пройти один раз.

### Веб-панель (GUI) — планировщик постов

```bash
python web.py
```

Открой в браузере **http://127.0.0.1:8000**.

1. **Первый экран — профили.** Можно привязать несколько Telegram-аккаунтов.
   Если профилей нет — жми «＋ Добавить аккаунт» и пройди GUI-вход
   (`api_id`/`api_hash` → телефон → код → пароль 2FA). Каждый профиль хранит
   свою сессию в папке `profiles/`.
2. **Выбери профиль** → откроется экран отправки.
3. **«＋ Добавить чат»** — поиск чатов как в Telegram (лс / группы / каналы).
   Можно выбрать сразу несколько — например, чтобы запостить в несколько каналов разом.
4. Напиши текст и задай **когда отправить**:
   - **конкретные даты** → отправка один раз в каждую из этих дат;
   - **только дни недели** → повтор каждую неделю в эти дни;
   - **ничего не задано** → повтор каждый день.
5. «Запланировать» — добавит правило (виден список активных расписаний),
   «Отправить сейчас» — отправит немедленно.

Расписания выполняет фоновый планировщик внутри `web.py` — он должен быть запущен
в нужное время (повторяющиеся отправки работают, пока сервер включён).
Файлы профилей и расписаний лежат в `profiles/` (`profiles.json`, `schedules.json`,
`*.session`) — это доступ к аккаунтам, **никому не передавай**.

### Терминал

```bash
python userbot.py
```

При первом запуске:

1. Получи `api_id` и `api_hash` на https://my.telegram.org → **API development tools**.
2. Скрипт спросит их, потом номер телефона.
3. Telegram пришлёт код → вводишь его в терминал (если включена 2FA — ещё пароль).
4. Создастся файл `user_session.session` — больше код вводить не нужно.

Дальше:

- Появится список чатов → выбираешь номер (или вводишь `@username` / ID).
- Видишь историю и входящие сообщения в реальном времени.
- Пишешь текст → он уходит в чат от твоего имени.

## Команды внутри чата

| Команда   | Действие                          |
|-----------|-----------------------------------|
| `/chats`  | вернуться к выбору другого чата    |
| `/history`| показать ещё сообщения истории    |
| `/quit`   | выход                             |

## ⚠️ Важно

- Это **userbot** — автоматизация личного аккаунта. Telegram разрешает это в разумных
  пределах, но за спам/массовую рассылку аккаунт могут **заблокировать**.
- Файлы `.credentials` и `*.session` — это доступ к твоему аккаунту.
  **Никому не передавай и не коммить в git** (уже в `.gitignore`).
